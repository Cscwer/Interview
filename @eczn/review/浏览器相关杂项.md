# 存储 

## Cookie 

客户端存储 http 状态。不能存很大的数据，数据存在客户端。 

存在一定的安全问题，所以不能存敏感数据

## Session

服务端存储 http 状态。


## LocalStroge 

前端键值对存储：字符串对字符串。 

可以用来存json数据。

现在工业上常用的前端存储方案。 


## indexDB 

前端数据库，接触的不多，工业上使用也不多。
